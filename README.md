# ConsultAccel: From Zero to High-Ticket Consultant

**ConsultAccel** is a comprehensive, 60-day interactive roadmap designed to transform aspiring consultants into high-ticket professionals. This application provides a structured, daily curriculum covering everything from foundational strategy to final service delivery.

---

## üöÄ Overview

The consulting world can be overwhelming. **ConsultAccel** simplifies the journey by breaking it down into 60 actionable days, organized into four strategic phases:

1.  **Phase 1: Foundation & Strategy (Days 1-15)**
    *   Market analysis, niche selection, and mindset shifts.
2.  **Phase 2: Offer Construction (Days 16-30)**
    *   Building irresistible high-ticket offers and value propositions.
3.  **Phase 3: Marketing & Lead Gen (Days 31-45)**
    *   Establishing authority and creating a consistent lead pipeline.
4.  **Phase 4: Sales & Delivery (Days 46-60)**
    *   Mastering sales calls and delivering exceptional client results.

## ‚öôÔ∏è Getting Started

### Prerequisites

-   [Node.js](https://nodejs.org/) (v18 or higher recommended)
-   npm or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/dineshmahapatra123/consulting-app.git
    cd consulting-app
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open the app**:
    Navigate to `http://localhost:5173` in your browser.

### Deployment

To deploy the app to GitHub Pages:
1. Ensure your `vite.config.js` has the correct `base` path (e.g., `'/consulting-app/'`).
2. Run the following command:
```bash
npm run deploy
```
*For a detailed step-by-step guide on authentication and initial setup, see [DEPLOYMENT_GUIDE.md](./DEPLOYMENT_GUIDE.md).*
